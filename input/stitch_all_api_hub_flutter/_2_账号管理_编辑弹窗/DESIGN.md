# Design System Principles

## Colors

### Brand Palette
Our design system utilizes a specific color palette to maintain brand consistency and visual appeal.

*   **Primary Color**: `#6750A4` - This is the brand's most distinctive chromatic color, ideal for prominent interactive elements like buttons and calls-to-action.
*   **Secondary Color**: `#625B71` - A versatile supporting color for less prominent UI elements, chips, and secondary actions, providing visual hierarchy.
*   **Tertiary Color**: `#7D5260` - An additional accent color used for highlights, badges, or decorative elements to enrich the visual experience.

### Color Mode
The system currently operates in `light` mode.

## Typography

### Font Families
We employ a consistent typographic scale across the application to ensure readability and brand identity.

*   **Headline Font**: `inter` - Used for all major headings and titles, providing strong emphasis.
*   **Body Font**: `inter` - Applied to all main text content for clear and comfortable reading.
*   **Label Font**: `inter` - Utilized for labels, small text, and UI annotations.

## Shape and Form

### Roundedness
Our UI elements feature a `moderate` level of roundedness (value `2`), striking a balance between modern aesthetics and approachability.

## Spacing

### Density
The design system maintains a `normal` level of spacing (value `2`), ensuring a balanced visual density that is neither too compact nor excessively spacious.
